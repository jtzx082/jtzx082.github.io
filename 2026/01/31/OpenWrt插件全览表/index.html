OpenWrt插件全览表

Posted on 2026-01-31 Edited on 2026-02-01

根据文件内容，我整理了其中包含的所有 luci-app 插件（共计 105 个），并按功能类别进行了分类和详细说明。

以下是完整的 OpenWrt 插件清单：
``

1. 科学上网、VPN与代理 (VPN & Proxy)

  • luci-app-openclash: 功能强大的 Clash 客户端，支持策略分流、规则订阅。
  • luci-app-passwall: 流行的代理客户端，支持 SS/SSR/V2Ray/Trojan 等多种协议。
  • luci-app-homeproxy: 新兴的家用代理工具，界面简洁。
  • luci-app-dae / luci-app-daed: 基于 eBPF 的高性能透明代理工具及其管理面板。
  • luci-app-v2raya: V2RayA 的 Web 界面，全功能的 V2Ray 管理工具。
  • luci-app-openvpn / luci-app-openvpn-server: OpenVPN 的客户端与服务器配置。
  • luci-app-ipsec-vpnd / luci-app-libreswan / luci-app-strongswan-swanctl: IPsec VPN 及其组件。
  • luci-app-zerotier: ZeroTier 虚拟局域网 SD-WAN 工具。
  • luci-app-softether / luci-app-softethervpn: SoftEther VPN 客户端与服务器。
  • luci-app-n2n: N2N 点对点 VPN 组网工具。
  • luci-app-ocserv: OpenConnect VPN (Cisco AnyConnect 兼容) 服务器。
  • luci-app-microsocks: 轻量级 SOCKS5 服务器。
  • luci-app-tor: Tor 匿名网络客户端。

2. 内网穿透与隧道 (Tunneling)

  • luci-app-frpc / luci-app-frps: FRP 内网穿透的客户端与服务端。
  • luci-app-nps: NPS 内网穿透工具。
  • luci-app-ngrokc: Ngrok 客户端。
  • luci-app-gost: GO Simple Tunnel，多协议安全隧道。
  • luci-app-sshtunnel: 建立 SSH 隧道。
  • luci-app-pagekitec: PageKite 反向代理客户端。
  • luci-app-natmap: NAT 端口映射/TCP 打洞工具。
  • luci-app-xfrpc: XFRP 内网穿透客户端。
  • luci-app-eoip: Ethernet over IP 隧道协议。

3. 广告拦截与访问控制 (Adblock & Access Control)

  • luci-app-adblock / luci-app-adblock-fast: 基于 DNS 的广告拦截工具（Fast 版本更轻量）。
  • luci-app-appfilter: 应用过滤（Open App Filter），可禁止特定 APP（如游戏、视频）联网。
  • luci-app-acl: 访问控制列表配置。
  • luci-app-banip: 批量封禁特定国家/地区或黑名单 IP。
  • luci-app-antiblock: 防屏蔽/域名转发工具。
  • luci-app-privoxy / luci-app-tinyproxy / luci-app-squid: 各种 Web 代理与缓存服务器（Privoxy 常用于去广告）。
  • luci-app-crowdsec-firewall-bouncer: 集成 CrowdSec 社区威胁情报的防火墙组件。

4. 网络管理与带宽 (Network Management)

  • luci-app-mwan3: 多宽带负载均衡与故障转移。
  • luci-app-syncdial: 多拨工具（并发拨号叠加带宽）。
  • luci-app-sqm: 智能队列管理，解决网络拥堵延迟（Bufferbloat）。
  • luci-app-qos / luci-app-eqos: 服务质量管理，限速与流量优先级控制。
  • luci-app-pbr: 策略路由 (Policy Based Routing)。
  • luci-app-upnp: 通用即插即用（自动端口映射）。
  • luci-app-wol / luci-app-timewol: 网络唤醒与定时唤醒。
  • luci-app-arpbind: IP/MAC 地址绑定。
  • luci-app-firewall: 防火墙设置（系统核心组件）。
  • luci-app-bcp38: 防止 IP 源地址伪造。
  • luci-app-apinger: 网络链路连通性监控。
  • luci-app-keepalived: 高可用性故障切换 (VRRP)。
  • luci-app-lldpd: 链路层发现协议守护进程（用于拓扑发现）。

5. DNS 与域名服务 (DNS & DDNS)

  • luci-app-smartdns: 智能 DNS 服务器，优化解析速度和结果。
  • luci-app-https-dns-proxy: DoH (DNS over HTTPS) 代理。
  • luci-app-unbound: 强大的递归/缓存 DNS 服务器。
  • luci-app-ddns / luci-app-ddns-go: 动态域名解析客户端（Go 版本界面更现代）。
  • luci-app-nextdns: NextDNS 官方客户端。

6. 下载、文件共享与存储 (Storage & Sharing)

  • luci-app-aria2: 全能下载工具（HTTP/FTP/BT）。
  • luci-app-transmission / luci-app-qbittorrent: 专业的 BT/PT 下载客户端。
  • luci-app-amule: 电驴 (eD2k/KAD) 下载客户端。
  • luci-app-xlnetacc: 迅雷快鸟/下载加速。
  • luci-app-samba4 / luci-app-ksmbd: Windows 文件共享 (SMB/CIFS) 服务端。
  • luci-app-vsftpd: FTP 服务器。
  • luci-app-nfs: NFS 网络文件系统。
  • luci-app-cifs-mount: 挂载远程 SMB 共享到本地。
  • luci-app-rclone: 挂载云存储（如 OneDrive, Google Drive）。
  • luci-app-minidlna: DLNA/UPnP 媒体服务器。
  • luci-app-dufs / luci-app-filebrowser / luci-app-filebrowser-go / luci-app-filemanager: 各种 Web 端文件浏览器。
  • luci-app-diskman: 磁盘管理工具（分区、格式化）。
  • luci-app-hd-idle: 硬盘自动休眠工具。

7. 系统运维与监控 (System & Monitoring)

  • luci-app-ttyd: 网页版终端 (Web SSH)。
  • luci-app-statistics: 系统性能统计图表。
  • luci-app-nlbwmon: 详细的内网主机带宽流量监控。
  • luci-app-vnstat2: 历史流量统计。
  • luci-app-netdata: 实时炫酷的性能监控面板。
  • luci-app-watchcat: 断网/故障自动重启（看门狗）。
  • luci-app-autoreboot / luci-app-advanced-reboot: 定时重启与高级重启。
  • luci-app-attendedsysupgrade: 在线固件升级助手。
  • luci-app-package-manager: 软件包管理器。
  • luci-app-cpulimit: 进程 CPU 占用限制。
  • luci-app-ramfree: 内存释放工具。
  • luci-app-irqbalance: 中断负载均衡（优化多核性能）。
  • luci-app-commands: 自定义 Shell 命令菜单。
  • luci-app-xinetd: 超级守护进程管理。
  • luci-app-snmpd: SNMP 服务端（用于远程监控）。
  • luci-app-log-viewer (隐含在 luci-base 或 openlist 中，列表中未直接出现单体包，但属于常用功能)。

8. Docker 与虚拟化 (Virtualization)

  • luci-app-docker: Docker 引擎配置。
  • luci-app-dockerman: Docker 容器 Web 管理界面。
  • luci-app-lxc: LXC 容器管理。

9. 无线与物联网 (Wireless & IoT)

  • luci-app-wifi-schedule: WiFi 定时开关。
  • luci-app-usteer: WiFi 漫游辅助 (802.11k/v/r)。
  • luci-app-dawn: 分布式 WiFi 管理器。
  • luci-app-travelmate: 旅行路由器工具（自动连接公共热点）。
  • luci-app-coovachilli / luci-app-openwisp: 热点认证网关与控制器。
  • luci-app-dcwapd: 无线 AP 控制器守护进程。
  • luci-app-mosquitto: MQTT 消息代理（智能家居常用）。
  • luci-app-lorawan-basicstation: LoRaWAN 网关配置。
  • luci-app-3ginfo-lite / luci-app-modemband / luci-app-sms-tool-js: 3G/4G/5G 模组信息、频段锁定与短信工具。
  • luci-app-dump1090: 接收飞机 ADS-B 信号。

10. 校园网与认证 (Campus Auth)

  • luci-app-sysuh3c: 中山大学 H3C 认证。
  • luci-app-scutclient: 华南理工大学认证。
  • luci-app-njitclient: 南京工程学院认证。
  • luci-app-bitsrunlogin-go: 深澜校园网认证。
  • luci-app-cd8021x: 802.1x 有线认证。
  • luci-app-ua2f: 修改 User-Agent（用于绕过部分校园网检测）。

11. 其它服务与工具 (Others)

  • luci-app-unblockneteasemusic: 解锁网易云音乐灰色歌曲。
  • luci-app-airplay2: 苹果 AirPlay 音频接收端。
  • luci-app-spotifyd: Spotify 音乐播放器。
  • luci-app-music-remote-center: 音乐遥控中心。
  • luci-app-wechatpush: 微信/钉钉状态推送。
  • luci-app-vlmcsd: KMS 激活服务器。
  • luci-app-udpxy / luci-app-msd_lite / luci-app-omcproxy: IPTV 组播转单播工具。
  • luci-app-usb-printer / luci-app-p910nd: USB 打印服务器。
  • luci-app-mjpg-streamer: USB 摄像头视频流服务。
  • luci-app-ser2net: 串口转网络（远程控制串口设备）。
  • luci-app-chrony: 精确的时间同步服务器。
  • luci-app-email: 邮件发送配置。
  • luci-app-clamav: 杀毒软件。
  • luci-app-radicale2: 卡片与日历同步服务器 (CardDAV/CalDAV)。
  • luci-app-rustdesk-server: RustDesk 远程桌面自建服务器。
  • luci-app-syncthing: 文件自动同步工具。
  • luci-app-pppoe-server / luci-app-rp-pppoe-server / luci-app-pppoe-relay: PPPoE 拨号服务器与中继。
  • luci-app-ps3netsrv: PS3 游戏网络加载服务。
  • luci-app-splash: 强制门户（Splash Page）。
  • luci-app-oscam: 数字电视读卡服务。
  • luci-app-3cat / luci-app-argon-config / luci-app-openlist: 主题配置与列表管理。
  • luci-app-babeld / luci-app-bmx7 / luci-app-olsr / luci-app-olsr-services / luci-app-olsr-viz: 各种 Mesh 动态路由协议。
